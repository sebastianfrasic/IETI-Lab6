# Lab # 6 - IETI
# 2.1 Task Planner API


### Pre-requisitos 📋

* Java 8
* Git
* Docker
* Gradle



### Instalación 🔧

1. Clonar el repositorio

```
git clone https://github.com/sebastianfrasic/IETI-Lab6.git
```


2. Para ejecutar la aplicación

```
npm install
```


```
npm start
```
